# Calculadora de IMC

App desenvolvido em React Native para calcular o Índice de Massa Corporal.
Foi programado na plataforma snack.expo.dev, tendo uma melhor execução nela.

## 📱 Funcionalidades

- Cálculo do IMC baseado em peso e altura
- Classificação do resultado (Magreza, Normal, Sobrepeso, Obesidade, Obesidade Grave)
- Validação de dados inseridos

## 🛠️ Tecnologias utilizadas

- React Native
- JavaScript
- Expo (se estiver usando)
